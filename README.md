Minecraft 2D - Beta 1.2
Welcome to Minecraft 2D! A fan-made 2D recreation of the classic sandbox experience.

🚀 How to Install
To ensure the game runs correctly, follow these steps to set up the file structure:

Extract the ZIP file: Right-click the downloaded ZIP folder and select "Extract All..."

Move to C: Drive: Move the extracted Games folder directly into your C: drive.

The final path must look like this: C:\Games\myMC\

Run the Game: Open the myMC folder and run the Minecraft2D.jar file.

Note: If the game fails to load images or music, please verify that the folder path is exactly C:\Games\myMC\ and that all files are inside their respective folders.

🎮 Controls
Move: Use 'A' (Left) and 'D' (Right) keys.

Jump: Press 'SPACE'.

Pause: Press 'ESCAPE' to open the menu.

Fullscreen: Toggle 'F11' to enter or exit full-screen mode.

Interact: Right-click to interact with the world.

🛠 Troubleshooting
"The game won't open": Ensure you have the latest version of Java installed on your computer.

"I don't hear music": Make sure the PARAMONA.wav file is present in the C:\Games\myMC\ folder.

Permissions: If images are not appearing, try right-clicking the .jar file and selecting "Run as Administrator."
